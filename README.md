# NeuroTunes

Premium enterprise therapeutic music platform by NeuralPositive — evidence-based,
patented technology that turns music into measurable therapeutic outcomes.

## Getting started

Requirements: Node.js & npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Enter the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the dev server with auto-reloading and an instant preview
npm run dev
```

## Tech stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the production bundle with `npm run build` and deploy the contents of `dist/`
to any static host or your chosen platform.
